Dungeon Project

"The Dungeon Project" is a game demo I created towards the end of 2021 as part of my job application process. I used free low-poly art assets to build this third-person action game framework demo. This demo included fundamental technological elements such as character control, camera animations, attack detection, network synchronization, and more.

The purpose of creating this project was to showcase my abilities in game programming. As a result, the gameplay features in this project are relatively basic, emphasizing the demonstration of my development skills.

Full game package available upon request.
